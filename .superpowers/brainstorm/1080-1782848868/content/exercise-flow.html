How should the "one exercise at a time" screen work?

You want to go through exercises sequentially, marking each complete to advance. How should a single exercise screen be laid out?

A

One set at a time

Screen shows ONE set: "Set 2 of 4". You log weight + reps for that set, tap "Done", it advances to the next set. After the last set, advances to the next exercise. Most granular — matches "log on each rep" closely.

B

One exercise at a time, all sets visible

Screen shows the exercise name + a small table/list of all its sets (e.g. 4 rows for 4×6-8). You fill in weight+reps per set as you go. One "Mark Exercise Complete" button advances to the next exercise.

C

Hybrid: exercise card with active set highlighted

Like B, but the current set is highlighted/expanded for quick entry, completed sets collapse into a compact row above. Swipe or tap to move between sets, auto-advances to next exercise when all sets done.
